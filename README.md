# Трекер прочитанных книг

Консольное приложение на Python для учёта прочитанных книг с возможностью ставить оценки, смотреть статистику и управлять библиотекой.

## Что делает программа

Программа помогает вести личный дневник прочитанных книг. Вы можете:
- Добавлять книги с оценкой и датой прочтения
- Просматривать весь список книг
- Смотреть среднюю оценку по всем книгам
- Получать статистику по каждому автору
- Удалять книги из списка

Все данные сохраняются в файл `books.json` и не теряются после перезапуска программы.

## Файлы и их назначение

### `main.py` - основной файл программы
Содержит весь код приложения. В нём есть:

#### Функции для работы с данными:
- `load_books()` - загружает книги из `books.json`
- `save_books(books)` - сохраняет книги в `books.json`
- `is_duplicate(books, author, title)` - проверяет, нет ли уже такой книги

#### Функции с бизнес-логикой:
- `add_book(books)` - добавляет новую книгу
- `show_all_books(books)` - показывает все книги
- `show_average_rating(books)` - считает среднюю оценку
- `show_author_stats(books)` - делает статистику по авторам
- `delete_book(books)` - удаляет книгу по номеру
- `main()` - главный цикл программы (меню)
